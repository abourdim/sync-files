❓ Help
What is SyncFiles?

A smart file sync tool for developers. Delta sync, conflict detection, Google Drive & Git integration.

How does conflict detection work?

If a file changes on both sides, SyncFiles creates a backup and alerts you. You choose which version to keep.

What is delta sync?

Like rsync — files split into chunks, only changed chunks transferred. Great for large files.

Is my data private?

Yes. Runs locally, credentials encrypted, dashboard localhost-only by default.

1

Run ./sync.sh --install to set up.

2

Edit config.yaml — set source and destinations.

3

Set up credentials: ./sync.sh --creds

4

Launch: ./sync.sh --web

⚡ Delta Sync

4MB chunks, SHA256 hashed, only changed chunks transferred.

⚠️ Conflicts

Both sides modified = conflict. Backup auto-created. Resolve in dashboard.

🔒 Credentials

Fernet AES-256 encryption, PBKDF2 key derivation, master password protected.

🔒 Privacy

Local-first. No tracking. Dashboard localhost-only. Your files, your control.
